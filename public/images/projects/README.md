# Project Images

Place your project screenshots in this directory with the following names:

- `heir-luxury.png` - Screenshot of the Heir Luxury e-commerce platform
- `portfolio.png` - Screenshot of this portfolio website
- `sales-app.png` - Screenshot of the Sales Tracking App

## Recommended image specifications:
- Format: PNG or JPG
- Aspect ratio: 16:9 (video aspect)
- Recommended size: 1920x1080px or 1280x720px
- Max file size: 500KB (optimize for web)

## Tips for great screenshots:
1. Capture the most visually interesting part of your application
2. Use high-quality, crisp screenshots
3. Consider adding browser chrome or device mockups for context
4. Optimize images before uploading (use tools like TinyPNG or Squoosh)
